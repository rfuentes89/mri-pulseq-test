# MRI Pulseq

Repositorio personal para aprendizaje y experimentación con **PyPulseq** y simulación de secuencias MRI.

## Objetivos
- Diseñar secuencias MRI con PyPulseq
- Entender restricciones de hardware (dead times, raster)
- Simular señal y k-space
- Evolucionar hacia secuencias clínicas (GRE, SE, EPI)

## Estructura
sequences/ # Diseño de secuencias Pulseq
simulation/ # Simulación de señal e imagen
utils/ # Funciones auxiliares


## Entorno
El entorno de Python se define en `environment.yml`.

## Notas
Este repositorio es educativo y experimental.
